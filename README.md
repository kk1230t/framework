
## UI Framework
웹 인터페이스 개발을 위한 UI Framework입니다.
보편적인 웹 개발에서 필요한 UI 모듈 대부분을 포함하고 있습니다.


## 사용 방법

### 1. 체크아웃
```
git clone https://github.com/kk1230t/framework.git
```

### 2. 저장폴더로 이동
```
cd framework
```

### 3. Install all dependencies
```
npm install
```
