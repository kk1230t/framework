웹 인터페이스 개발을 위한 UI Framework입니다. 보편적인 웹 개발에서 필요한 UI 모듈 대부분을 포함하고 있습니다.



모듈 리스트

sticky, dropdown, grid, collapse, customToggle, slick, tab, scrolltab, selectbox, modal, window popup, scrollspy, scrollItem, moreview, starRating, tooltip, video

selector

ui스크립트 모듈에 사용되는 셀렉터는 모두 data-modules-[모듈이름]을 사용합니다.

셀렉터 및 옵션 값 작성 요령 - data-modules-[모듈이름] 속성을 이용한 호출


	

	
	

		

			

			
<div data-modules-moduleName="property:value;property:value;property:value;......"></div>


			
옵션값 지정은 css style 작성 방식과 동일하게 값을 넣어주시면 됩니다. 전달 할 옵션 값이 없을 경우 <div data-modules-moduleName></div> 까지만 선언


			
			

				
					
					
					
				
				
					

						Value Type
						example
						description
					

				
				
					

						Boolean
						true, false
						
					

					

						String
						string...
						따옴표 없이 전달.
					

					

						Number
						1, 2, 33, 100, -10
						따옴표 없이 전달.
					

					

						selector
						'.classname', '#idname'
						홀따옴표 주의.
					

					

						json data
						{'el1':'value', 'el2':'value',....}
						속성 명과 값 모두 홀따옴표 지정 주의
					

				
			

		

	

	
셀렉터 및 옵션 값 작성 요령 - 함수 직접호출

	

		

			
직접 호출시에는 셀레겉로 어떤 요소를 이용해도 무방함.

			

				$(element).moduleName({

					      el:value,

					      el1:value,
