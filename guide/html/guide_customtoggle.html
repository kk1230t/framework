Customtoggle

selector

[data-modules-customtoggle]

function

$(element).customtoggle()

options

Name Type default description
activeClass string ui-active 토글 시킬 클래스 이름
textCtr selector [data-text] 셀렉트 토글과 함께 텍스트 문구도 변경해야 할 경우 사용, 이 셀렉터는 기본 셀렉터 하위에 위치해야 합니다.
parent selector null 토글 시킬 셀렉터, 기본은 자신입니다.
openText String 더보기 클래스 토글과 함께 변경할 문구가 있을 시, activeClass 클래스 요소가 활성화 되면 적용됩니다 ! 셀렉터 하위에 textCtr 요소가 있을 경우
closeText String 닫기 클래스 토글과 함께 변경할 문구가 있을 시, activeClass 클래스 요소가 제거되면 적용됩니다, ! 셀렉터 하위에 textCtr 요소가 있을 경우
closeBtn selector [data-close-btn] 커스텀토글 닫기 버튼

Method

Name description
$(element).customtoggle('toggle') 토글 메서드

Event

Name description
$(element).on('toggle.customtoggle') 토글 이벤트
