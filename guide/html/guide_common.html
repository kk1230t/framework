commonUi

selector

$('body')

function

$('body').commonUi()

이 함수는 "DOMContentLoaded" 이벤트 발생 시 기본으로 호출됩니다.

페이지에서 사용하는 전체 모듈을 일괄 로드시키며, 사이트 전체에서 사용하는 기능이 정의되어 있는 함수입니다.
