Intro

페이지 스크롤 시 화면에 선택된 요소를 고정 시켜주는 기능

html문서 내 호출

	
	<!-- 기본 -->
	<div data-modules-sticky>
		마크업 내용...
	</div>

	<!-- 커스텀 옵션 사용 -->
	<div data-modules-sticky="padding:0;relativeTarget:#wrap;">
		마크업 내용...
	</div>

script에서 호출

	
	// 기본 옵션없이 호출
	$('SelectorName').sticky();


	// 커스텀 옵션 사용
	$('SelectorName').sticky({
		padding:0,
		relativeTarget : '#wrap'
	});

최소한의 CSS

	
[data-modules-sticky] .sticky-inner{

}
[data-modules-sticky].fixed{
	position: fixed;
}

options

Name Type default description
padding Number 0 기본으로는 화면상단에 붙으면 작동, 수치 조절을 통해 작동 범위를 조절 할수 있음
breakPoint selector null 전달된 셀렉터의 위치 밑으로는 내려가지 않음.
className String null 초기화 시 sticky 셀렉터를 <div class="sticky-ui-wrapper"></div>로 랩핑하는데 이때 랩핑되는 부모요소에 삽입될 클래스 명
