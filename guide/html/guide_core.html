Intro

Framework 전역에서 사용 및 접근할수있는 객체에 대한 설명입니다.
브라우저 정보, 해상도 정보 등의 데이터와 각종 유틸리티 함수등을 포함하고 있습니다.

선언


	// 전역 네임스페이스에서 선연
	var CoreUi = UI;
	console.log(CoreUi) 	// {name: "UI", DEFAULT_SCREEN_SIZE: Array(3), version: "0.7.0", noop: ƒ, emptyFn: ƒ, …}

	또는...

	//즉시실행 함수에서 선언
	;(function(core, $, undefined){

		console.log(core);	// {name: "UI", DEFAULT_SCREEN_SIZE: Array(3), version: "0.7.0", noop: ƒ, emptyFn: ƒ, …}

	})(window[LIB_NAME], jQuery);

1. 전역객체에서 제공하는 데이터

1) 전역객체에서 제공하는 데이터

property description
UI.DEFAULT_SCREEN_SIZE script에서 사용하는 해상도별 분기 Array 입니다.
UI.detect 사용하는 브라우저의 정보를 포함하고 있습니다. (브라우저 종류, 버전, 사용하는 OS 등..)
UI.prefix event namespace 문자열입니다.(.hui)
UI.userScreenControl 사용자가 화면을 컨트롤하는지 체크하는 Boolean 입니다.(scroll 제외 모듈 사용시 체크)

2-1. Selector - 태그에 삽입하여 호출

ui스크립트 모듈에 사용되는 셀렉터는 모두 data-modules-[모듈이름]을 사용합니다.

	
	<!-- 모듈 사용 예:기본 옵션 없이 호출) -->
	<div data-modules-sticky>
		마크업 내용...
	</div>

	<!-- 모듈 사용 예:커스텀 옵션 사용) -->
	<div data-modules-sticky="padding:0;relativeTarget:#wrap;">
		마크업 내용...
	</div>

2-2. Selector - <script>에서 직접 호출

스크립트로 직접 호출하는경우 data-modules-[모듈이름]에서 "모듈이름" 부분이 함수명이 됩니다.

	
	// 기본 옵션없이 호출
	$('SelectorName').sticky();


	// 커스텀 옵션 사용
	$('SelectorName').sticky({
		padding:0,
		relativeTarget : '#wrap'
	});

3. 옵션 값 작성 가이드

Value Type example description
Boolean true, false
String string... 따옴표 없이 전달.
Number 1, 2, 33, 100, -10 따옴표 없이 전달.
selector .classname, #idname
json data {'el1':'value', 'el2':'value',....} 속성 명과 값 모두 홀따옴표 지정 주의
